# Kotlin S1E2

## Kotlin vs Lombok

### Kotlin features

- Java 19+ features
    - basic language
    - var
    - records
    - sealed class
    - pattern matching*
    - ...
- **Lombok features**
    - val
    - data class
    - extension functions*
    - delegation*
    - ...
- Kotlin specific features
    - coroutines
    - DSL (Domain Specific Language)
    - inline/infix/reified functions
    - property delegation
    - contracts
    - ...
